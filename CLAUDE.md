# ICS_DMAS 專案規則

## 安全與供應鏈規則

**禁止使用任何與中國相關的軟體、函式庫、或服務。**

包含但不限於：
- npm / PyPI 套件的維護者或主要貢獻者為中國實體
- 中國雲端服務（阿里雲、騰訊雲、百度雲等）
- 任何來源不明、無法驗證供應鏈的套件

遇到不確定的情況，**必須先詢問使用者確認**，不得擅自引入。

## 開發環境

- 主要開發機：Mac
- 目標部署：Raspberry Pi（未到貨，到貨後驗證）
- 測試方式：Mac 本機模擬完整架構

## 專案結構

- `command-dashboard/` — 指揮部後端（FastAPI + SQLite）
- `shelter-pwa/` — 收容組 PWA + WebSocket Pi 伺服器
- `medical-pwa/` — 醫療組 PWA（開發中）
