### 1-3 收容流程中違禁品管制

> 專用版編號：收容[普]21

| 欄位 | 內容 |
|------|------|
| 發生場域 | 收容所收容登記區、物資存放區 |
| 機密層級 | C |
| 事件描述 | 收容階段中需要檢查收容者是否攜帶違禁品，將要求收容者自己攤開攜帶物品來目視檢查，並以金屬探測器掃描收容者 |
| 人力需求 | 維安人員一名 |
| 資源需求 | 安檢桌一張、燈具兩副、金屬探測器一把、箱子數枚、膠帶兩捲、簽字筆一隻、管制表格、維安人員隨身配備一套 |

**【整備階段】**

1. 配合人管隊收容流程在登記區內架設安檢桌
2. 收容過程中請收容者將物品攤出檢查
3. 收容人員以金屬探測器掃描收容者
4. 在持有者面前將違禁物品置於箱子內並寫上物品持有者姓名編號後封箱，並送往物資存放區保存
5. 填寫管制表格：物品名稱及數量、持有者名稱及編號、是否已退還、退還時持有者簽名

**【演訓階段】**

| 步驟 | 說明 | 配合組別 |
|------|------|---------|
| 1 | 於登記區內架設安檢桌 | — |
| 2 | 收容流程中請收容者將物品攤出、以金屬探測器掃描收容者 | — |
| 3 | 將違禁品裝箱、標記、打包後送置物資存放區保管（若有疑慮可請收容者簽名於封口處） | — |
| 4 | 收容所撤收時於出口處連箱歸還物品（非緊急撤離） | — |

對應緊急事件：**收容者不願配合違禁品管制**

**【場復階段】**

1. 配合收容流程，收容流程結束時將安檢桌移置安檢區內，維安人員回歸環管隊分配事項

### 1-4 收容區環境清潔

> 專用版編號：收容[普]22（另有通用版「計畫[普]01：收容區環境清潔」，內容相同）
> 
> 差異說明：專用版使用「環管小隊」稱呼，通用版使用「環管組成員」；步驟說明文字略有差異，但流程一致。

| 欄位 | 內容 |
|------|------|
| 發生場域 | 收容所 |
| 機密層級 | C |
| 事件描述 | 演訓期間，環管成員依照隊長指示，在收容區內進行環境清掃和收集回收垃圾，任務完成後回報組長，以便支援協助其他組成員 |
| 人力需求 | 環管成員一名 |
| 資源需求 | 掃把畚箕拖把抹布各一、大型垃圾桶 3+ 個、大小垃圾袋各一組、酒精一瓶 |

**【整備階段】**

1. 根據活動行程表並抵達活動場景，清點身上裝備和執行職務時所需物品配備與人力
2. 配合隊長指示，進行已分配好的任務，分別清潔整理收容所場地
3. 巡邏並收集分類好垃圾後回報給組長，然後待命以便調派支援人力資源

**【演訓階段】**

| 步驟 | 說明 | 配合組別 |
|------|------|---------|
| 1 | 開幕典禮結束後，開始盤點整理自身裝備與人力資源並前往任務目的地 | 收容組、環管隊 |
| 2 | 抵達收容區後，做好場佈並依據事先分配好的工作計畫書進行分工 | 收容組、環管隊 |
| 3 | 進行任務期間中，因應現場環境調整策略，並且用無線電定期回報組長狀況以確保工作在期限內完成 | 收容組、環管隊 |
| 4 | 分工任務完成後，回報給組長了解近況並回到指定目的地待命，以便組長調派人力資源支援其他組員 | 收容組、環管隊 |

對應緊急事件：**發現可疑人士或危險物品**、**場地設備遭破壞**

**【場復階段】**

1. 清點環管組的各項工具與自己裝備品項，確認有無缺少或損毀，以便收容組和環管組分辨紀錄回報上層
2. 確認收容區場地狀況，進行清掃回收垃圾，將場地恢復到場佈前模樣

### 緊急事件：收容區場地或設備遭破壞

> 專用版編號：收容[緊]19

| 欄位 | 內容 |
|------|------|
| 發生場域 | 收容所 |
| 機密層級 | C |
| 通報流程 | 環管隊員 → 環管隊長 → 收容組組長 → 副指揮 → 指揮官 |
| 事件描述 | 由於收容區因外力（如地震洪水、敵軍來襲或轟炸），使收容所整體結構不穩甚至失去安全性，因而轉移下一個安全區 |

| 步驟 | 說明 | 配合組別 |
|------|------|---------|
| 1 | 聽到緊急廣播或發現收容區已有建築開始有土石掉落，甚至是有破片飛濺的時候，先找安全的地方避難，並用無線電回報環管隊長自身狀況和尋求指示｜資源：無線電 | 環管隊 |
| 2 | 得到下一個指示及許可後，以能夠保自身安全下僅攜帶自身所需的裝備工具，而其餘工具直接放置在不佔據逃生通道的其他空地上，注意自身環境並直接前往集合地點｜資源：無線電 | 環管隊 |
| 3 | 抵達集合地點後，再次確認自身身體健康狀況和裝備工具是否遺漏，回報環管隊長清點組員人數，完成後開始轉移下一個安全區｜資源：無線電 | 環管隊 |
| 4 | 轉移到安全區時，確認自身和其他組員身體狀況跟裝備工具，然後靜候指示並待命｜資源：無線電 | 環管隊 |

對應普通事件：**收容區環境清潔**

### 2-4 盤點收補：剩餘物資如何回收

> 通用版編號：收容物管[普]盤點收補 02

| 欄位 | 內容 |
|------|------|
| 發生場域 | 收容所物資作業區、垃圾放置區 |
| 機密層級 | C |
| 事件描述 | 基於衛生安全以及避免資源浪費等考量，配發給民眾的物資若有剩餘，僅回收符合收容所實際需求物資（以全新無使用過之物資為優先） |
| 人力需求 | 物管小隊 |
| 資源需求 | 剩餘物資回收清單、原子筆、板夾、移入庫存單、麥克筆、「未拆封物資退還區」標示板 |

**【整備階段】**

1. 確立能回收的物品，僅回收符合收容所實際需求物資（以全新無使用過為優先）
2. 不能回收的物品，引導民眾自行至垃圾放置區處理
3. 回收物資經檢查確認安全衛生無虞後，視情況選擇移回庫存或其他運用
4. 移回庫存的部分需配合即時更新庫存盤點數量

**【演訓階段】**

| 步驟 | 說明 | 配合組別 |
|------|------|---------|
| 1 | 設立「物資退還/交換站」：於物資組旁設置標註「未拆封物資退還區」的明顯區域，並請民眾填寫剩餘物資回收清單；鼓勵民眾將不需用或規格不符之完整包裝物資帶至退還區 | 物管隊 |
| 2 | 依據包裝完整度與衛生風險分三類：（1）可回收：包裝完整無破損、無沾染髒污、保存期限尚可；（2）待判定：外包裝完整、效期不明或較短、無衛生安全疑慮（由隊長、組長負責最終判定）；（3）不回收：依垃圾放置區分類原則處理 | 物管隊、物管隊隊長、收容組組長 |
| 3 | 可回收物資視情況選擇移回庫存或其他運用，填寫剩餘物資回收清單與移入庫存單。注意：效期緊迫度；可回收物資需在庫存單上註記為經核可之剩餘物資（無論是否已移入庫存） | 物管隊、物管隊隊長、收容組組長 |
| 4 | 搬運與盤點注意：需要簽收；若有交接設定，必須確實完成盤點再進行交接；新進夥伴未交接時請先進行盤點；視情況不定時進行盤點，確保物資狀態及在保存期限內 | 物管隊、物管隊隊長、收容組組長 |

對應緊急事件：**非判定可回收之物資誤入庫存**

**【場復階段】**

1. 彙整剩餘物資回收清單與移入庫存單，確認表單內容是否有所缺漏
2. 盤點尚未移入庫存中的物資（包含借用公物是否都已回到物管隊？是否有過期或不堪使用之物資？）
3. 視情況汰除過期或不堪使用之物資；非消耗型特殊物資（如輪椅、體溫計）經整理或清潔完畢後，歸還至原主/原單位/捐贈予收容民眾；消耗型或其他特殊物資依自身限制與情況個別處理
4. 原子筆、麥克筆、板夾等相關文具回收
