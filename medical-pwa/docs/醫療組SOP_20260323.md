# 一、醫療組任務定位與編組

## 1.1 任務目標

醫療組為據點內之醫療支援單位，主要任務如下：

1.  管理醫療資源：統計人員、器材、藥品，評估並掌握醫療量能。

2.  開設醫療站：於據點內建立檢傷區、治療區（紅、黃、綠）、後送區，執行緊急醫療處置。

3.  傷患管理：接收、檢傷、處置、紀錄、後送。

4.  維持醫療區安全：確保醫療區無武裝威脅，必要時請安全組支援。

5.  回報指揮部：定期回報傷患數、醫療量能、重大狀況。

## 1.2 核心原則

**安全優先順序：**自身安全 → 夥伴安全 → 患者安全。這是最優先原則。

- 未經解除武裝者，不予治療。
- 醫療可暫停，人員安全不可妥協。
- 先救可救者，不糾結於單一患者。
- 持續再評估，隨時調整處置優先順序。

## 1.3 人員編組（19人配置）

以下為建議編組，各單位可依實際人力調整：

| **職務** | **人數** | **主要職責** |
|---|---|---|
| 組長 | 1 | 總指揮、醫療量能評估、對指揮部回報 |
| 副組長 | 1 | 協助組長、代理指揮、監控各區運作 |
| 通訊官 | 1 | 對外聯絡指揮部、協調後送、回報傷患數量與量能 |
| 後勤官 | 2 | 醫療物資管理、藥品補給、器材盤點、發放登記 |
| 檢傷官 | 2 | 執行START檢傷分類、區分創傷/非創、分流患者至各治療區 |
| 紅區治療官 | 4 | 處理危急傷患（立即威脅生命）、XABCDE/ABC評估與處置 |
| 黃區治療官 | 4 | 穩定傷患、持續觀察、防止惡化 |
| 綠區治療官 | 2 | 輕傷處置、心理支持、協助綠區患者自理 |
| 後送官 | 2 | 準備轉送、執行ISBAR交接、與外部運輸協調 |
| 機動 / 輪替 | 1 | 支援各區、接替疲勞人員、緊急從動 |

**特別說明：**人員配置可依實際狀況彈性調整，例如綠區傷患多時可從機動人員支援；紅區傷患多時可從黃區抽調人手。組長同時需隨時掌握各區狀況並調度人力。

# 二、醫療站設站流程

## 2.1 目的

在據點內建立一個安全、有組織、可持續運作的醫療站，以接收、檢傷、處置與後送傷患。

## 2.2 設站流程（逐步操作）

### 第一步：選址

團隊到場後，立即評估據點內可用空間，選定醫療站位置。選址原則：

- 遠離威脅區域（不靠近外牆、武裝區）
- 具備可控制的單一入口與單一出口
- 空間足夠容納檢傷區、治療區、後送區、物資堆放和作業區
- 照明充足有水有電
- 確保後送動線順暢

### 第二步：劃分區域

依照「三、醫療區區域配置規範」劃定各區，以旗幟、指示牌、桌椅標示明確分隔。

### 第三步：建立單向動線

所有患者從「入口」進入，經過安檢、檢傷、治療後，從「出口」離開或後送。禁止逆向流動。

### 第四步：建立通訊

通訊官建立與指揮部的聯絡管道（無線電、對講機、手機），確認頻道與呼號。

### 第五步：分配人員與整備物資

組長宣布各人職務與位置。後勤官將物資分配至各區。確認各區裝備到位。

### 第六步：組長最終確認與開站

- 確認各區就位
- 確認通訊暢通
- 確認裝備到位
- 向指揮部回報：「醫療站開設完畢，可以收治傷患」

# 三、醫療區區域配置規範

## 3.1 配置原則

- 單一入口：所有傷患由此進入，安全組在入口執行安檢。
- 單一出口：處置完畢之輕傷患者由此離開；後送患者亦由出口方向接駁後送車輛。
- 單向動線：入口 → 安檢 → 檢傷 → 治療區 → 後送區/出口。禁止逆向流動。
- 各區標示清楚（紅、黃、綠旗幟或標誌）。

## 3.2 區域說明

### ① 入口安檢區

安全組駐守，對所有進入醫療區人員執行安檢（脫除武裝、通訊設備、危險物品）。未完成安檢者不得進入醫療區。

### ② 檢傷區（Triage Area）

檢傷官駐守。執行START檢傷分類，同時區分患者為創傷或非創傷，以決定後續評估流程（創傷走XABCDE、非創走ABC）。檢傷官負責分類，除需要先打止血帶，原則不做治療。

### ③ 紅區（Immediate）

收治立即威脅生命之傷患。配置最多人力與裝備，採 XABCDE 或 ABC 流程評估處置。

### ④ 黃區（Delayed）

收治可延遲處置但需持續觀察之傷患。定時再評估，若惡化則升級至紅區。

### ⑤ 綠區（Minor）

收治輕傷、可自行行動之患者。可協助自理（如自行加壓止血），提供心理支持。處置完畢後經出口離開。

### ⑥ 黑區（Expectant / Deceased）

無生命跡象或依當前醫療量能無法救治者。放置於靠近出口但與其他區域分開之處。待資源允許時再評估。

### ⑦ 後送區

靠近出口。將經評估需後送之傷患移至此區等待運輸。後送官駐守，負責 ISBAR 交接與與運輸單位協調。

### ⑧ 物資區

後勤官管理。存放備用醫療器材、藥品、耗材。各區需要補給時向後勤官申請。

# 四、醫療區標準運作流程

## 4.1 目的

描述醫療站從開設到關閉的完整運作週期，讓每一個人知道「現在該做什麼」。

## 4.2 運作週期

### 階段一：開設期

依「二、設站流程」執行。完成後向指揮部回報開設完畢。

### 階段二：運作期

1.  持續接收傷患（依「五、接收傷患流程」）。

2.  執行檢傷、處置、後送（依「六、急救處置流程」）。

3.  組長每 30 分鐘向指揮部回報一次醫療量能。

4.  後勤官每 30 分鐘盤點一次物資。

5.  組長安排人員輪替休息（建議每 2 小時換班一次）。

**醫療量能回報格式（範例）：**

「指揮部，醫療組回報：目前紅傷 3 名、黃傷 5 名、綠傷 8 名、已後送 2 名。可用紅區床位剩 2床、止血帶剩 6 條。」

### 階段三：交班期

若任務持續超過 4 小時，應執行交班。交班採用 ISBAR 格式：

- I（Identify）：我是誰、接班人是誰
- S（Situation）：目前各區狀況（紅幾人、黃幾人、綠幾人）
- B（Background）：重要背景（例如「紅區二號床患者有大出血史」）
- A（Assessment）：當前評估（例如「醫療量能即將飽和」）
- R（Recommendation）：建議（例如「建議優先後送紅區一號床」）

**交班重點：**紅區患者需逐一交接，不可只做口頭概述。交班前不可離開崗位。

### 階段四：關站期

1.  組長下達關站指令。
2.  停止收治新傷患。
3.  完成現有患者處置或後送。
4.  後勤官清點全部物資。
5.  組長向指揮部回報：「醫療站關閉，紅傷 X、黃傷 X、綠傷 X、後送 X、死亡 X」。

# 五、接收傷患流程

## 5.1 目的

確保每一位進入醫療區的傷患都經過安全檢查、身份登記、檢傷分類，並依傷勢分流至正確區域。

## 5.2 傷患來源與接收方式

傷患可能來自三種管道，每種接收方式略有不同：

### A. 前進組搜救後送達

1.  前進組將傷患帶至入口。
2.  安全組在入口執行安檢（脫除武裝、通訊設備、危險物品）。
3.  **前進組以 MIST 進行口頭交接：**
    - M（Mechanism）：受傷機轉（例：建物倒塔壓傷、爆炸波及）
    - I（Injuries）：已發現傷害（例：左下肢開放性骨折、頭部擦傷）
    - S（Signs）：目前生命徵象（例：HR 120、BP 90/60、RR 24、GCS 13）
    - T（Treatment）：已做處置（例：已上止血帶、已給氧）
4.  檢傷官接手，重新執行 START 檢傷分類（不可僅依賴前進組判斷）。
5.  若患者尚未登記身分，通知收容組同步登記。
6.  依檢傷結果分流至對應治療區。

### B. 收容組轉送

1.  收容組將需要醫療協助的災民送至醫療區入口。
2.  安全組執行安檢（即使已經過收容組登記，仍需安檢）。
3.  檢傷官執行 START 檢傷分類。
4.  確認收容組已登記身分，若未登記則通知收容組補登。
5.  依檢傷結果分流。

### C. 自行抵達醫療區

1.  傷患自行來到入口。
2.  安全組執行安檢。
3.  檢傷官執行 START 檢傷分類。
4.  通知收容組登記身分（因自行抵達者通常未被登記）。
5.  依檢傷結果分流。

## 5.3 情況危急時的例外處理

若傷患情況危急（例如大出血、失去意識），可直送醫療區評估，但必須同步執行以下三件事：

1.  安全組在入口執行安檢（同時進行，不等待）。
2.  安全組驗證後綁上手環。
3.  收容組同人登記身分。

**重要：**即使情況危急，安檢仍然是必要的。差別在於「同時執行」而非「省略」。

## 5.4 接收後的三種處置方向

每位傷患經過評估處置後，會有三種可能的結果：

1.  **處置後離開：**輕傷患者（綠區）經處置後，确認無惡化風險，由出口離開醫療區。
2.  **留置觀察：**黃區患者或有惡化風險者，留在醫療區內定時再評估。
3.  **後送醫療院所：**紅區患者或超出醫療量能者，依「八、患者轉送流程」後送。

# 六、醫療區急救處置流程

## 6.1 目的

統一醫療區內的急救評估與處置流程，依傷患類型選擇對應的評估架構。

## 6.2 創傷與非創區分

檢傷官在 START 分類時，同時判定患者是「創傷」或「非創」：

- 創傷: 有明顯外力造成之傷害。
- 非創: 因疾病造成之生理不適。

## 6.3 創傷患者處置流程：XABCDE

這是創傷醫學標準流程，每一步都要「發現問題就立刻處理」，再往下一步。

### X -- Exsanguination（致命出血控制）

**看什麼：**有沒有大量出血？血流噴出、大片血漬、肢體截肢？

**怎麼做：**

- 直接加壓止血
- 必要時上止血帶（四肢）
- 使用止血敷料進行填塞（交界處）

### A -- Airway（呼吸道）

**看什麼：**患者能說話嗎？有没有打鼾？口內有異物嗎？

**怎麼做：**

- 壓額抬下巴（無頸椎損傷疑慮時）或進行下顎推舉（有頸椎損傷疑慮時）
- 清除口內異物
- 置入 OPA/NPA 維持呼吸道
- 若有頸椎損傷疑慮，維持頸椎固定

### B -- Breathing（呼吸）

**看什麼：**呼吸速率、深度、對稱性。胸壁有傷口嗎？有氣胸跡象嗎？

**怎麼做：**

- 給氧（非再吸入式面罩 / BVM）
- 開放性氣胸：胸封貼貼上
- 張力性氣胸：針刺減壓
- BVM 輔助通氣

### C -- Circulation（循環）

**看什麼：**脈搏（速率、強弱）、膚色（蒼白發紺）、CRT（指尖微血管充填）、有無休克跡象？

**怎麼做：**

- 控制出血（再次檢查）
- 建立 IV/IO 通路、給予輸液
- 保暖（防止低體溫加劇休克）

### D -- Disability（神經評估）

**看什麼：**GCS 評分、瞳孔大小與對光反射、四肢活動力。

**怎麼做：**

- 評估並記錄 GCS
- 若意識不清，評估血糖

### E -- Exposure（暴露與保護）

**看什麼：**全身有沒有其他傷害？需要剪開衣服檢查嗎？

**怎麼做：**

- 必要時剪開衣物檢查
- 檢查完畢後立即覆蓋保暖毯
- 防止低體溫（創傷死亡鑽石：低體溫、酸中毒、凝血功能異常、低血鈉）

## 6.4 非創患者處置流程：ABC + 輔助評估

適用於無明確受傷機轉的患者（內科急症、慢性病發作等）。

### A -- Airway（呼吸道）

同創傷流程，確保呼吸道暢通。

### B -- Breathing（呼吸）

評估呼吸狀況，必要時給氧或 BVM 輔助通氣。

### C -- Circulation（循環）

評估脈搏、血壓、膚色。若無脈搏無呼吸，啟動 CPR + AED。

### 輔助評估

- SAMPLE 病史：Symptoms（症狀）、Allergies（過敏）、Medications（用藥）、Past history（病史）、Last meal（最後進食）、Events（事件經過）
- OPQRST 疼痛評估：Onset（發作）、Provocation（誘因）、Quality（性質）、Radiation（放射）、Severity（嚴重度）、Time（時間）
- 評估血糖（若有設備）
- 心電圖監測（若有設備）

## 6.5 共通處置原則

1.  每位患者都必須填寫救護紀錄表。
2.  持續再評估：危急每 5 分鐘、非危急每 15 分鐘。
3.  若患者狀況變化，重新檢傷分類（黃升紅或綠升黃）。
4.  不要長時間卡在單一患者，大量傷患時先做救命處置再繼續下一位。

# 七、患者送醫判斷原則與 START 檢傷分類

## 7.1 START 檢傷分類流程

START（Simple Triage and Rapid Treatment）是大量傷患事件中最常用的檢傷分類系統。以下是完整的判斷流程：

**【 患者可以自行行走嗎？ 】**

**→ 可以 → 🟢 綠色**

**→ 不可以 → 繼續往下評估**

**【 檢查呼吸 】**

觀察呼吸：

**→ 呼吸小於30下 → 繼續評估**

**→ 呼吸大於30下 → 🔴 紅色**

**→ 打開呼吸道後才有呼吸 → 🔴 紅色**

**→ 沒呼吸 → ⚫ 黑色**

**【 檢查循環 】**

觸摸橈動脈（或檢查指尖微血管充填 CRT）：

**→ 有橈動脈且CRT<2秒 → 繼續評估**

**→ 沒有橈動脈或CRT>2秒 → 🔴 紅色**

**【 檢查意識 】**

給予簡單指令（例如「請你握一下我的手」）：

**→ 可以配合 → 🟡 黃色**

**→ 無法配合 → 🔴 紅色**

## 7.2 紅區與黃區的判斷依據整理

| **🔴 紅色（Immediate）-- 立即威脅生命** | **🟡 黃色（Delayed）-- 可延遲但需觀察** |
|---|---|
| 無法行走 + 呼吸異常（>30） | 無法行走，但呼吸正常 |
| 無橈動脈 或 CRT > 2秒 | 有橈動脈 且 CRT ≤ 2秒 |
| 無法聽從簡單指令 | 可以聽從簡單指令 |

## 7.3 送醫判斷原則

### 必須後送（紅區）

- 呼吸道無法維持或持續惡化
- 大量出血未能有效控制
- 休克徵象且現場處置無法改善
- GCS 持續下降
- 需要手術介入的傷害（開放性骨折、穿刺傷等）

### 建議後送（黃區）

- 骨折（尤其長骨骨折）
- 內出血疑慮（腹部壓痛、瘀傷擴大）
- 中度頭部外傷
- 燒燙傷面積較大
- 需影像檢查確認的傷害

### 可留置觀察（綠區）

- 輕微擦傷、切割傷、瘀傷、撕裂傷
- 可自行行動，生命徵象穩定

### 判斷流程

**【 是否立即危及生命？ 】**

**→ 是 → 立即後送**

**→ 否 → 是否超出現場醫療量能？**

**→ 是 → 後送**

**→ 否 → 留置觀察，定時再評估**

# 八、患者轉送流程

## 8.1 目的

將超出現場醫療量能或需要進一步醫療處置的傷患，安全、有效率地轉送至後方醫療院所。

## 8.2 轉送前提

**重要觀念：**轉送的前提不是「患者完全穩定」，而是「已完成必要的救命處置」。

意思是：你已經做了你在現場能做的事（止血、維持呼吸道、給氧、建立通路），接下來患者需要醫院才能提供的處置（手術、輸血、檢查CT等），就該送。不要因為「感覺還沒穩定」而延誤後送時機。

## 8.3 轉送流程（逐步操作）

1.  **治療官判斷需要後送：**依「七、送醫判斷原則」判斷。
2.  **通知後送官：**「後送官，紅區 X 號床需要後送！」
3.  **後送官通知通訊官：**通訊官聯絡指揮部協調運輸車輛 / 救護車。
4.  **治療官完成救護紀錄：**確保所有處置已記錄。
5.  **移至後送區：**將患者移至後送區等待。
6.  **後送官執行 ISBAR 交接：**與接收單位（救護車 / 醫院）交接。

**ISBAR 交接內容：**

- I（Identify）：「我是 XX 民防某據點醫療組後送官」
- S（Situation）：「現有一名紅區傷患需要轉送」
- B（Background）：「患者因建物倒塌壓傷，左下肢開放性骨折」
- A（Assessment）：「目前 HR 120、BP 90/60、GCS 14，已止血、已給氧、已建立 IV」
- R（Recommendation）：「建議優先送至有重大創傷手術能力之醫院」

7.  **通訊官回報指揮部：**「已後送一名紅傷至 XX 醫院」。
8.  **後勤官更新醫療量能紀錄。**

## 8.4 後送優先順序

1.  紅區傷患（最優先）
2.  黃區傷患（依病況）
3.  綠區傷患（最後）

**注意：**黑區患者不後送（除非資源允許且有醫療意義）。

# 九、醫療物資領用流程

## 9.1 目的

確保醫療資源可控、可追蹤、不浪費，並優先供應最需要的區域。

## 9.2 物資管理責任

後勤官為物資管理唯一窗口。所有物資領用必須經過後勤官。

## 9.3 領用流程

1.  各區治療官提出需求（口頭或書面）。
2.  後勤官確認庫存是否足夠。
3.  發放物資並登記（誰領、領什麼、多少）。
4.  後勤官每 30 分鐘盤點一次庫存。
5.  關鍵物資即將耗盡時，立即向組長回報，由組長向指揮部申請補給。

## 9.4 管制物資清單

以下物資應列入管制，優先供應紅區：

| **類別** | **物資項目** | **管制原因** |
|---|---|---|
| 止血類 | 止血帶、止血敷料、壓力繃帶、紗布 | 消耗快、救命關鍵 |
| 呼吸道類 | OPA/NPA、BVM、氧氣瓶、面罩 | 數量有限、無法補充 |
| 循環類 | IV&IO、生理食鹽水、葡萄糖溶液 | 數量有限、需特殊訓練 |
| 藥品類 | 止痛藥、抗生素 | 需注意過敏、劑量、效期 |
| 固定類 | 軟式護木、三角巾、頸圈 | 骨折處置必要 |
| 保暖類 | 保暖棉被、鋁箔毯 | 防止低體溫 |
| 診斷類 | AED、血氧機、血糖機、血壓計 | 器材有限、電池需確認 |
| 紀錄類 | 救護紀錄表、檢傷卡、筆 | 法律與紀錄需求 |
| 防護類 | 手套、口罩、護目鏡、C防隔離衣 | 感控防護 |

# 十、危害醫療區安全應變指引

## 10.1 目的

當醫療區發生安全威脅時，統一的應變流程，確保人員安全。

## 10.2 可能的威脅情境

- 患者或家屬情緒激動、暴力行為
- 未經安檢人員闖入
- 發現傷患攜帶武器或危險物品
- 外部威脅接近據點

## 10.3 應變流程

1.  **發現威脅：**任何人發現異常狀況。
2.  **立即呼叫：**大聲喊「保安！保安！」，通知安全組介入。
3.  **暫停醫療：**所有醫療人員停止處置，後退至安全位置。
4.  **保護自身：**不要試圖自行制服威脅人員，讓安全組處理。
5.  **安全組控制現場：**安全組介入處理威脅、帶離威脅人員。
6.  **組長評估：**確認威脅解除後，決定「恢復運作」或「啟動撤退」。
7.  **回報指揮部：**通訊官向指揮部回報事件經過與處置結果。

## 10.4 特殊患者處理原則

### 精神不穩定 / 暴力傾向患者

- 安全組全程在場
- 指派專責人員處理
- 必要時隱離其他患者
- 優先評估是否需後送

### 傳染病疑慮患者

- 立即隱離
- 處置人員穿戴 PPE（手套、口罩、護目鏡）
- 通知組長與指揮部

# 十一、醫療區撤退疑散流程

## 11.1 目的

當醫療區無法繼續安全運作時，依序撤離所有人員與傷患。

## 11.2 啟動撤退的時機

- 指揮部下達撤退命令
- 醫療區遭受無法控制的安全威脅
- 建物結構損壞有倒塌風險
- 組長判斷繼續運作將危及人員安全

## 11.3 撤退流程（逐步操作）

1.  **組長下達撤退命令：**大聲宣布「醫療區撤退！」，或依單位規定使用哨音信號。
2.  **立即停止收治新傷患：**入口關閉，不再接收。
3.  **快速穩定現有患者：**做最後一次快速評估，確保患者可以安全移動。
4.  **優先撤離紅區患者：**由後送官協調，紅區患者優先後送或撬離。
5.  **依序撤離黃區、綠區患者：**綠區可自行行走者先撤離。
6.  **後勤官收拾關鍵物資：**優先攜帶：AED、藥品、氧氣。其他物資視時間決定是否攼帶。
7.  **醫療人員撤離：**確認所有患者已撤離後，醫療人員方可撤離。
8.  **組長最後撤離：**組長確認無人後，最後離開。
9.  **向指揮部回報：**「醫療區已完成撤退，人員全數安全」。

## 11.4 撤退優先順序

1.  紅區傷患（最優先）
2.  黃區傷患
3.  綠區傷患（可自行移動者先行）
4.  黑區患者（依資源判斷）
5.  醫療物資
6.  醫療人員
7.  組長（最後）

**絕對原則：**不可遺留任何傷患。若無法攼帶黑區患者，必須向指揮部回報並註記位置。
